## Guiding Principles

> *The objective is not to build the largest system. The objective is to build the most enduring one.*

This system is designed to:
- Preserve what was created, why, how it evolved, and what was learned
- Enable discovery and serendipitous connection of ideas
- Support emergence (one insight → research → lecture → paper → book → social content)
- Compound over time (each project improves the next)
- Outlast platforms and technologies

---

## Architecture

**Permanent Truth**: GitHub (version control, discoverability, longevity)

**AI Partners**: Claude (architecture, documentation) + ChatGPT (strategy, creativity)

**Creative Pipeline**: Capture → Research → Insight → Markdown → GitHub → Visual Design → Publishing → Analytics → Reflection

**Publishing**: Adobe Express + Firefly (design) → Buffer (scheduling) → All platforms

**Measurement**: GA4 + analytics → Reflection → System improvement
