# PURPOSE

The Tao of Clinical Touch is not marketed like a massage business.

It is presented like an editorial publication.

Every visual should communicate quiet confidence, scientific credibility, therapeutic alliance, and physiological safety.

Our images do not sell techniques.

They communicate relationships.

They communicate trust.

They communicate permission.

They communicate possibility.

The visual language should immediately distinguish The Tao of Clinical Touch from every massage therapy brand online.

When someone sees one of our images, they should recognize the brand before reading a single word.

---

# BRAND PHILOSOPHY

## Core Principle

> Stillness communicates competence.

The strongest clinician is not the busiest clinician.

The strongest clinician creates an environment where change becomes possible.

Every photograph should feel like it captured a real moment inside a world-class clinical practice.

Nothing feels posed.

Nothing feels marketed.

Nothing feels performative.

---

# EMOTIONAL HIERARCHY

Every image should communicate these emotions in order:

1. Safety
2. Presence
3. Trust
4. Permission
5. Possibility
6. Clinical Confidence

Never reverse this order.

Technique is never the first message.

# COMPOSITION

Photographs should feel intimate.

Viewer should feel positioned beside the treatment table.

Hands become the primary storytelling element.

Large empty treatment rooms are discouraged.

Clinical clutter should disappear.

Background should support—not compete.

# LIGHTING

Only natural light.

Preferred:

- Soft overcast daylight
- Warm reflected daylight

- Hard shadows
- Blue lighting
- Colored lighting

- High contrast
- Overhead fluorescent appearance

# RIPPLE WATERMARK

This becomes our anti-theft signature.

NOT a logo.

NOT branding.

NOT decoration.

A permanent editorial watermark.

Specifications:

- Uses canonical ripple artwork from book cover

- Cropped by canvas

Every.

Single.

One.

# CREATIVE REVIEW CHECKLIST

Before approving any image, Claude should verify:

- [ ] Does this communicate trust before technique?
- [ ] Does the therapist appear calm rather than forceful?
- [ ] Does the patient appear safe rather than passive?
- [ ] Is the environment clinical rather than spa-like?
- [ ] Are the hands listening rather than imposing?
- [ ] Is the ripple watermark present?
- [ ] Is the correct chapter symbol used?
- [ ] Is the composition intimate?
- [ ] Does the image avoid stock-photo aesthetics?
- [ ] Would this still look elegant five years from now?

If any answer is "No," regenerate the image.

# IMPLEMENTATION NOTES FOR CLAUDE

## When generating Firefly prompts
Use the Master AI Image Prompt above as the foundation. Customize the specific story/emotion but maintain the structural requirements (chapter symbol, ripple watermark, lighting standards, environment type).

## When creating Adobe Express templates
Reference the Platform Templates section. Maintain consistent margins, typography, and placement of chapter symbols and watermarks across all platform variations.

## When reviewing existing assets
Run through the Creative Review Checklist. If any element fails, mark for regeneration.

## When encountering new platforms
Apply the Master Prompt structure, adapt dimensions to platform specs, and maintain all core visual requirements (chapter symbol, ripple watermark, emotional hierarchy, photography standards).

## Version control
This is v1.0. Future versions should only be created when execution evidence (from campaigns, analytics, user feedback) demonstrates material limitations in this system. Per the Governance Principle, aesthetic preferences are not sufficient reason to revise.
