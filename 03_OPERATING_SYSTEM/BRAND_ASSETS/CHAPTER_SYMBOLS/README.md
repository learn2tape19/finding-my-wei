## Usage Standards

### Placement
- Upper left corner of image (never centered or repositioned)
- Approximately 3% of image width
- White or Linen color (#f9f8f7 or #f5e6d3)
- Consistent margin from edges (never touching border)
- Clean background for visibility (never overlaid on complex area)

### Technical Specifications
- Format: SVG only (never convert to PNG/JPG)
- Dimensions: 100×100px viewBox (scales to any size)
- Stroke width: 1.6 (fixed, never change)
- Stroke linecap: round
- No fill (outline only)
- Color: White or Linen only
- Never rotate, flip, distort, or redesign

### Integration with Firefly Prompts
Reference in Master + Specific section:
```
Include canonical [SYMBOL NAME] chapter symbol in upper-left corner.
Integrate canonical Tao ripple watermark as large subtle watermark (5–8% opacity).
```

### Adobe Express Integration
Place as overlay element:
- Layer above photograph
- Fixed position (upper left, 3% from edge)
- Opacity: 100%
- No shadow or effects
- Size: Scale proportionally to image dimensions
