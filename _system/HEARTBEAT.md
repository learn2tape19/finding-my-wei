# HEARTBEAT

## PURPOSE
Drive daily execution by aligning TODAY with active project priorities.

## DAILY PROCESS

1. Review PROJECT_CONTROL files:
   - SidekickAir
   - Learn2Tape
   - Tao

2. Identify:
   - Highest leverage task per project
   - Any urgent blockers

3. Generate TODAY.md:
   - Define FOCUS
   - Select top 3 priorities
   - Define AVOID list

4. Check alignment:
   - Are tasks tied to outcomes?
   - Are we avoiding low-value work?

5. End of Day:
   - Log progress in project files or logs
   - Adjust PROJECT_CONTROL if priorities changed

## RULES
- No more than 3 priorities per day
- Priorities must move a project forward
- Avoid starting new work unless required
