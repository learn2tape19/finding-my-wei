## Evaluation Matrix Scoring

**Total Score = (Lightweight × 0.20) + (Folding × 0.20) + (Engineering × 0.15) + (Prototype × 0.15) + (Manufacturing × 0.10) + (Medical × 0.10) + (Collaboration × 0.05) + (Geographic × 0.05)**

**Score 0-100 in each category, then apply weights.**
