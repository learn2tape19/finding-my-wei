## Strategic Positioning

### What Makes This Defensible
1. **Honest about origins** — Organizing existing knowledge, not discovering new science
2. **Progressive credibility** — Each paper builds foundation for next
3. **Clear boundaries** — Explicit about what framework IS and IS NOT
4. **Practical focus** — Designed for clinician use, not academic abstraction
5. **Testable components** — Measurable outcomes identified
6. **Falsifiable** — Research questions identify what would disprove claims

### What Makes This Impactful
1. **Fills a real gap** — Alliance in manual therapy under-researched
2. **Addresses practitioner need** — Clinicians want better reasoning tools
3. **Foundation for future work** — Becomes architecture for other investigations
4. **Cross-modality** — Applicable across PT, massage, chiropractic, etc.
5. **Education potential** — Can be taught and systematized
6. **Research agenda** — Generates specific, testable questions
