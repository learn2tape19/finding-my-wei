## Finding My Wei Media Operating System v1.0

## INTRODUCTION

This checklist is required before ANY content is published.

Use this to:
- Ensure consistent quality
- Verify alignment with Editorial Constitution
- Catch errors before publication
- Document review process
- Create accountability

**Process:** Content doesn't publish until all boxes are checked.

---

## PHASE 1: RESEARCH REVIEW

**Owner:** Content Director (Claude)  
**Timeline:** Before any content finalization  
**Frequency:** Once per piece

### Research Quality

- [ ] Every factual claim has evidence OR is labeled (observation/hypothesis)
- [ ] Sources are recent (within 10 years for research)
- [ ] Sources are credible (peer-reviewed > gray literature > anecdotal)
- [ ] Study limitations are acknowledged
- [ ] Contradictory evidence is presented
- [ ] No cherry-picking (only confirming data shown)
- [ ] Statistics are accurate and match source
- [ ] Mechanisms are explained accurately

### Evidence Hierarchy

- [ ] High-confidence claims match high-quality evidence
- [ ] Moderate-confidence claims acknowledge limitations
- [ ] Low-confidence claims are labeled as hypothesis/emerging
- [ ] No "proven" claims for correlational data
- [ ] No absolute statements ("always," "never") without universal evidence

### Originality & Attribution

- [ ] Original ideas are identified as such
- [ ] Others' frameworks/research are credited
- [ ] No plagiarism of language or structure
- [ ] Paraphrases include attribution
- [ ] Direct quotes include page numbers
- [ ] Competing/alternative views acknowledged
- [ ] Proper citations in required format

### Accuracy Check

- [ ] Names spelled correctly
- [ ] Dates verified
- [ ] Titles match actual publications
- [ ] Statistics double-checked
- [ ] Technical terms used correctly
- [ ] No false claims or exaggerations
- [ ] Clinical examples are realistic

## PHASE 2: FACT REVIEW

**Owner:** Fact-Check Role  
**Timeline:** After research review  
**Frequency:** Once per piece

### Scientific/Medical Accuracy

- [ ] Medical claims are accurate per authoritative source
- [ ] Anatomy is described correctly
- [ ] Physiology is represented accurately
- [ ] No misleading health claims
- [ ] Disclaimers included if needed ("Not medical advice")
- [ ] Risks/side effects are mentioned
- [ ] Alternative approaches acknowledged

### Data & Statistics

- [ ] Numbers match original sources
- [ ] Units are correct (percentages, absolute numbers, etc.)
- [ ] Trends are represented accurately
- [ ] Outliers are identified and explained
- [ ] Sample sizes noted where relevant
- [ ] Margins of error/confidence intervals included
- [ ] Comparisons are appropriate (not comparing incompatible data)

### Citations & References

- [ ] Complete citations for all sources
- [ ] Citation format is consistent
- [ ] Links work (for online sources)
- [ ] No misquotes
- [ ] Context of quotes is accurate
- [ ] References section complete

### No Prohibited Content

- [ ] No unsubstantiated health claims
- [ ] No guaranteed results ("will heal," "guarantees")
- [ ] No harmful medical advice
- [ ] No promotion of pseudoscience
- [ ] No conflict of interest hidden
- [ ] No misleading comparisons
- [ ] No false urgency or artificial scarcity

**Sign-off:** Fact-Checker _____________________ Date _______

## PHASE 3: MESSAGE REVIEW

**Owner:** Editorial Director (ChatGPT)  
**Timeline:** After research and fact review  
**Frequency:** Once per piece

### Core Message Clarity

- [ ] Main idea is clear
- [ ] Purpose is obvious (why this content exists)
- [ ] Target audience is identified
- [ ] Value proposition is evident ("why should I read/care?")
- [ ] Key takeaways are memorable
- [ ] Call to action is appropriate
- [ ] Message doesn't get lost in details

### Audience Appropriateness

- [ ] Content matches audience knowledge level
- [ ] Respect for audience expertise is evident
- [ ] No oversimplification that sacrifices accuracy
- [ ] No unnecessary jargon
- [ ] Length is appropriate for topic
- [ ] Format matches audience platform preferences
- [ ] Examples are relatable

### Tone & Voice

- [ ] Professional but human
- [ ] Appropriate confidence level for evidence
- [ ] Consistent with Finding My Wei voice
- [ ] Conversational but credible
- [ ] No arrogance or false certainty
- [ ] No attacks on colleagues/competitors
- [ ] Respectful disagreement (if applicable)

### Structure & Flow

- [ ] Opening answers "why should I care?"
- [ ] Context is provided before core idea
- [ ] Ideas flow logically
- [ ] Transitions connect sections
- [ ] Conclusion ties back to introduction
- [ ] Key points stand out
- [ ] No important information buried
- [ ] Appropriate use of white space

### Persuasiveness

- [ ] Argument builds logically
- [ ] Evidence supports conclusions
- [ ] Counterarguments are addressed
- [ ] Emotional appeals are honest (not manipulative)
- [ ] No overselling or hype
- [ ] Invitation to thinking, not command
- [ ] Reader feels respected, not lectured

**Frequency:** Once per piece

- [ ] Design is professional and clean
- [ ] Color scheme matches brand guidelines
- [ ] Typography is readable (font size, contrast, typeface)
- [ ] Layout is balanced and organized
- [ ] No clutter or unnecessary elements
- [ ] Hierarchy guides reader attention
- [ ] White space is used effectively
- [ ] Brand elements are consistent

### Images & Graphics

- [ ] Images support message
- [ ] Images are high quality

- [ ] Captions formatted for platform

- [ ] Visual style matches Finding My Wei identity

- [ ] Professional and appropriate for audience

## PHASE 5: PLATFORM ADAPTATION REVIEW

**Owner:** Publishing Director (Buffer)  
**Timeline:** After visual review  
**Frequency:** Once per platform

### Platform-Specific Formatting

**For LinkedIn:**
- [ ] Opening line is compelling hook
- [ ] Formatting uses line breaks for readability
- [ ] Word count 800-1200 words
- [ ] Hashtags are relevant (3-5)
- [ ] Call to action is included
- [ ] No link shorteners (use full URL)
- [ ] Preview text is strong

**For Instagram:**
- [ ] Image quality is high (Instagram optimized)
- [ ] Aspect ratio correct (1.91:1 for feed, 9:16 for stories)
- [ ] Text on image is readable at phone size
- [ ] Caption under 150 words before "...more"
- [ ] Hashtags placed after caption (10-30 relevant tags)
- [ ] Call to action is clear
- [ ] Carousel slides are numbered

**For Website:**
- [ ] URL is SEO-friendly (keywords included)
- [ ] Title includes primary keyword
- [ ] Meta description is compelling
- [ ] H1 and H2 headers are optimized
- [ ] Internal links to related content
- [ ] External links to credible sources
- [ ] No broken links
- [ ] Mobile-responsive design
- [ ] Author bio included

**For Email/Newsletter:**
- [ ] Subject line is compelling (not spam-like)
- [ ] Sender name is recognizable
- [ ] Preview text is strong
- [ ] Mobile formatting is correct
- [ ] CTA buttons are clear and clickable
- [ ] Unsubscribe link present (required)
- [ ] No images-only format (text version exists)
- [ ] Send time is optimized

**For Social Media General:**
- [ ] No spelling/grammar errors
- [ ] Links are tested and working
- [ ] Images load properly on platform
- [ ] @ mentions are correct
- [ ] No broken formatting (symbols, emojis, line breaks)
- [ ] Timezone is correct for posting
- [ ] Scheduling is accurate

### Content Optimization

- [ ] SEO keywords are naturally integrated
- [ ] Readability level is appropriate
- [ ] Scannability is good (headings, bullets, short paragraphs)
- [ ] CTA is clear and platform-appropriate
- [ ] No platform-mixing (Instagram voice ≠ LinkedIn voice)
- [ ] Engagement potential is high (ask questions, invite comments)

## PHASE 6: ETHICS & INTEGRITY REVIEW

**Owner:** Editor-in-Chief  
**Timeline:** Before final approval  
**Frequency:** Once per piece

### Conflicts of Interest

- [ ] Conflicts of interest disclosed (if any)
- [ ] Promotion labeled as promotion (if included)
- [ ] Products/services mentioned are fairly represented
- [ ] No undisclosed advertising or sponsored content
- [ ] Personal financial interest is transparent
- [ ] Employment/affiliation disclosed if relevant

### Respect & Fairness

- [ ] Competitors/alternative approaches treated fairly
- [ ] Colleagues given credit for ideas
- [ ] No attacks on people (criticism of ideas only)
- [ ] Opposing viewpoints represented accurately
- [ ] Tone is respectful even when disagreeing
- [ ] No strawman arguments
- [ ] Good-faith interpretation of others' positions

### Confidentiality

- [ ] No patient information disclosed
- [ ] Patient stories are unidentifiable
- [ ] No employee/organizational secrets shared
- [ ] Colleagues' personal information protected
- [ ] Private conversations not quoted without permission
- [ ] Business details protected appropriately

### Legal & Compliance

- [ ] No defamation or false claims about anyone
- [ ] No copyright infringement
- [ ] Proper licenses for images/music used
- [ ] Fair use criteria met (if applicable)
- [ ] No privacy violations
- [ ] Medical disclaimers included (if needed)
- [ ] Terms/affiliate disclosure present (if needed)

### Ethical Boundaries

- [ ] No manufactured controversy or outrage
- [ ] No manipulation of emotions
- [ ] No exploitation of vulnerable audiences
- [ ] No false urgency or artificial scarcity
- [ ] No spread of misinformation
- [ ] Honest representation of alternative views
- [ ] Good intent is assumed (about others' motives)

## TROUBLESHOOTING

### Content Fails Research Review
→ Return to authors for revisions or additional research  
→ Loop back through Phases 1-3 when revised

### Content Fails Fact Review
→ Correct factual errors before proceeding  
→ If major revisions needed, return to Phase 1

### Low Engagement Despite Quality
→ Analyze: Wrong audience? Wrong platform? Wrong timing?  
→ Use insights to inform future content  
→ Consider test variations on next publication

### Negative Feedback
→ Assess: Is criticism valid? Do we need to clarify?  
→ Respond professionally, don't dismiss  
→ Update content if needed  
→ Learn for next time
