## CONTENT APPROVAL WORKFLOW

```
PHASE 1: RESEARCH REVIEW
├─ Owner: Content Director (Claude)
├─ Timeline: Before finalization
├─ Sign-off Required: Content Director
├─ Red Flag → Re-do research

PHASE 2: FACT REVIEW
├─ Owner: Fact-Checker Role
├─ Timeline: After Phase 1
├─ Sign-off Required: Fact-Checker
├─ Red Flag → Correct facts

PHASE 3: MESSAGE REVIEW
├─ Owner: Editorial Director (ChatGPT)
├─ Timeline: After Phase 2
├─ Sign-off Required: Editorial Director
├─ Red Flag → Rewrite for clarity

PHASE 4: VISUAL REVIEW
├─ Owner: Creative Director (Firefly)
├─ Timeline: After Phase 3
├─ Sign-off Required: Creative Director
├─ Red Flag → Redesign

PHASE 5: PLATFORM ADAPTATION
├─ Owner: Publishing Director (Buffer)
├─ Timeline: After Phase 4
├─ Sign-off Required: Publishing Director
├─ Red Flag → Reformat

PHASE 6: ETHICS & INTEGRITY REVIEW
├─ Owner: Editor-in-Chief
├─ Timeline: Before final approval
├─ Sign-off Required: Editor-in-Chief
├─ Red Flag → Major revisions or non-publication

PHASE 7: FINAL EDITORIAL APPROVAL
├─ Owner: Editor-in-Chief
├─ Timeline: Before publishing
├─ Sign-off Required: Editor-in-Chief
├─ Decision Options:
│  ├─ GO: Approve for publishing
│  ├─ HOLD: Request revisions, loop back to phase
│  └─ NO-GO: Return for major rework

PHASE 8: POST-PUBLICATION MONITORING
├─ Owner: Publishing Director
├─ Timeline: First 24-48 hours
├─ Sign-off Required: Publishing Director
├─ Watch For: Technical errors, corrections needed

PHASE 9: WEEKLY ANALYTICS
├─ Owner: Publishing Director
├─ Timeline: Weekly for 4 weeks
├─ Sign-off Required: Publishing Director
├─ Track: Engagement, performance, audience response

PHASE 10: MONTHLY REFLECTION
├─ Owner: Editorial Director
├─ Timeline: Monthly
├─ Sign-off Required: Editorial Director
├─ Output: Learning documentation, system improvements

PHASE 11: QUARTERLY ARCHIVE
├─ Owner: Editorial Director & Editor-in-Chief
├─ Timeline: End of quarter
├─ Sign-off Required: Editor-in-Chief
├─ Output: Archived content, integrated learning
```

## ESCALATION PATHS

```
QUESTION TYPE                    WHEN TO ESCALATE        TO WHOM
───────────────────────────────────────────────────────────────────

Accuracy of Content              Editorial standards      Content Director
                                 don't resolve

Message Clarity                  Conflicting             Editorial Director
                                 recommendations

Visual Design                    Conflicting             Creative Director
                                 visions

Platform Optimization            Technical               Publishing Director
                                 challenges

Conflict of Interest             Standards              Editor-in-Chief
                                 unclear

Ethical Boundary                 Any doubt              Editor-in-Chief

Publishing Dispute               No precedent           Editor-in-Chief
                                 available

Controversial Content            Values conflict        Editor-in-Chief

Resource/Timeline Issues         Capacity problem       Editor-in-Chief

System/Process Change            Effectiveness          Editor-in-Chief
                                 questions


ESCALATION PROCESS:
1. Identify the question type
2. Check if Editorial Standards address it
3. Look for precedent in past decisions
4. Check Editorial Constitution
5. If still unclear → Escalate to appropriate director
6. If director-level uncertain → Escalate to Editor-in-Chief
7. Editor-in-Chief makes final decision
8. Document decision & use as precedent going forward
```
