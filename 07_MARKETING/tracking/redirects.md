- [ ] Open clean public URL in incognito
- [ ] Confirm redirect lands on `/book/`
- [ ] Confirm UTM parameters visible in browser URL bar
- [ ] Confirm GA4 page_view event fires
- [ ] Confirm GTM event fires
- [ ] Click Amazon link
- [ ] Confirm `amazon_click` event fires
- [ ] Confirm event appears in GA4 DebugView
- [ ] Record result in `redirect_log.md`
- [ ] Mark `Tested: Yes` in this file
