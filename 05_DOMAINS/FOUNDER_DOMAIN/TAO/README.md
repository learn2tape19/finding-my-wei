# Welcome

This directory contains the complete intellectual domain for **The Tao of Clinical Touch**.

The Tao is not simply a book.

It is an evolving body of scholarship exploring the science, philosophy, and practice of therapeutic change through human touch.

Books, continuing education, research, speaking, Project Atlas, journal publications, and professional movements are all expressions of this single intellectual domain.

This directory preserves the ideas that generate those expressions.

# Adding New Work

Before adding a document ask:

1. Does this expand the Tao?

2. Does it belong to one of the established domain expressions?

3. Does it contribute enduring intellectual value?

4. Would future contributors benefit from preserving it?

If not, it likely belongs elsewhere.
